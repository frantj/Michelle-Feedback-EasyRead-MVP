To open and edit:

  1. Find the downloaded .html file in your Downloads folder
  2. Right-click the file and choose "Open With":

    • LibreOffice
  3. For Google Docs:
    • Upload the file to Google Drive
    • Right-click the file in Drive
    • Choose "Open with → Google Docs"
